# 🌍 Google 搜索链接生成器

一个简单的网页工具，用于生成带地区与指定网站参数的 Google 搜索链接。

## 功能
- 输入关键词
- 可选指定网站（如 twitter.com 或 .jp）
- 可选搜索地区（JP、US、TW、HK、SG 等）
- 一键生成可点击的搜索链接

## 使用方法
1. 打开 `index.html` 文件
2. 输入关键词（例如：`地震`）
3. 选择地区（例如：`日本`）
4. 可选输入网站（例如：`twitter.com`）
5. 点击“生成搜索链接”，即可获得一个可直接点击的 Google 搜索结果链接

## 在线部署建议
- 可直接上传到 **GitHub Pages** 或 **Netlify**
- 文件名：`index.html`
